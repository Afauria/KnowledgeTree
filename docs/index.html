Introduction

  1. clone项目
  2. gitbook install安装插件
  3. gitbook serve本地运行Gitbook，会打包到_book夹中，添加到.gitignore
  4. gitbook build . docs打包到docs文件夹中，用于部署到GitHub Pages中
  5. GitHub上Settings-Pages里面配置，选择docs目录，即可访问https://afauria.github.io/<仓库名称>/，由于绑定了域名，因此也可以访问https://blog.afauria.xyz/GitBook-Learner/
